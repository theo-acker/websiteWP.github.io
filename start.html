Guide rapide

Par où commencer ?

Suivez ces étapes pour lancer votre projet. Chaque carte est entièrement modifiable dans start.html.

01

Configurez votre environnement

Créez un dépôt GitHub et activez GitHub Pages depuis les paramètres du repo (onglet Pages).

02

Personnalisez les fichiers

Modifiez index.html, start.html et apropos.html selon vos besoins. Adaptez style.css à votre identité.

03

Ajoutez votre contenu

Remplacez les textes, images et couleurs. Intégrez des API externes si besoin via JavaScript.

04

Publiez sur GitHub Pages

Poussez vos fichiers sur la branche main. Votre site sera accessible à l'adresse username.github.io/repo.
